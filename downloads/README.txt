Konjac Gene Explorer v1.2.1 downloads

Files in this folder:

- Amorphophallus_konjac.clean.gff
  Gene structure / coordinate annotation.

- Amorphophallus_konjac.clean.cds
  CDS FASTA records.

- Amorphophallus_konjac.clean.pep
  Protein FASTA records.

- zen_annotation_v2.0.tsv
  ZEN functional annotation and homolog information.

- zen_go_v2.0.tsv
  GO annotation.

- zen_goslim_v2.0.tsv
  GO slim annotation.

- zen_interpro_v2.0.tsv
  InterPro domain annotation.

- zen_pfam_v2.0.tsv
  Pfam domain annotation.

This folder is intended for download by website users. The searchable website data are in ../data/genes.json.

Sources retained in the website metadata include PlantGARDEN t78372.G001, ZEN annotation v2.0, NCBI assembly GCA_022559845.1, GO, KEGG/KO/EC, InterPro, Pfam, and PlantTFDB.

Formal DOI, citation text, and license terms are pending. Candidate genes in the website are produced from annotation-based filtering and should not be treated as experimental validation.
