# Local BLAST assets

This directory is generated by `scripts/build-blast-db.ps1`.

Inputs:
- `downloads/Amorphophallus_konjac.clean.cds`
- `downloads/Amorphophallus_konjac.clean.pep`
- `data/processed/jbrowse-app/assemblies/GCA_022559845.1_ASM2255984v1_genomic.fna.bgz` when genome BLAST is enabled

Generated database prefixes:
- `blastdb/konjac_cds` (`nucl`)
- `blastdb/konjac_pep` (`prot`)
- `blastdb/konjac_genome` (`nucl`, local only, ignored by Git)

Run from the project root:

```powershell
.\scripts\build-blast-db.ps1
.\scripts\run-blast-examples.ps1
```

To include whole-genome `blastn` support for the Supabase worker:

```powershell
.\scripts\build-blast-db.ps1 -IncludeGenome
```

When the genome input is BGZF/GZIP, the script writes `blastdb/konjac_genome_source.fna` first because `makeblastdb` needs uncompressed FASTA. That local source FASTA and `konjac_genome.*` database files are ignored by Git.

The website remains static. These files support local command-line BLAST usage and future server-side BLAST integration.
