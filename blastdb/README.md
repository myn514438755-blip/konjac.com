# Local BLAST assets

This directory is generated by `scripts/build-blast-db.ps1`.

Inputs:
- `downloads/Amorphophallus_konjac.clean.cds`
- `downloads/Amorphophallus_konjac.clean.pep`

Generated database prefixes:
- `blastdb/konjac_cds` (`nucl`)
- `blastdb/konjac_pep` (`prot`)

Run from the project root:

```powershell
.\scripts\build-blast-db.ps1
.\scripts\run-blast-examples.ps1
```

The website remains static. These files support local command-line BLAST usage and future server-side BLAST integration.
